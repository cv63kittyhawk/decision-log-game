# 意思決定ログ可視化ゲーム

単一HTMLで作成したゲームプロトタイプです。

## 概要

プレイヤーが複数の意思決定シーンで選択を行い、その選択履歴（ログ）をもとに思考傾向を可視化するゲームです。
正解を当てることが目的ではなく、「どのように判断したか」というプロセスを観察・分析することに重点を置いています。
プレイ後には、思考タイプの診断結果とともに、選択の傾向が可視化されます。

## 遊び方

1. ページを開く
2. 各シーンで提示される選択肢を選ぶ
3. すべての質問に回答すると結果が表示される

## 目的

このプロトタイプでは以下を検証することを目的としています。

* 意思決定プロセスをゲームとして成立させられるか
* プレイヤーの選択ログから思考傾向を可視化できるか
* 診断結果だけでなく「過程」を見せることで体験価値が向上するか
* テキストベースの軽量構造でどこまでゲーム性を持たせられるか
